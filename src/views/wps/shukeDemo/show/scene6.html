1.打开版式页面，并在插件准备好后调用addTrackInfo或者setBarcodeInfo将水印或条码信息设置到阅读器插件。


首页 + 文字水印 + 显示 + 打印


首页 + 文字水印 + 不显示 + 打印


1,2页 + 文字水印 + 显示 + 不打印


首页 + 二维码 + 显示 + 打印


1-3页 + 文字水印旋转120度+背景覆盖



2.调用openFile或者openSvcFile打开文件后，预定位置后将出现水印或者条码。以openFile为例。



3.打开文档后可进行的水印操作。


末页 + 文字水印 + 显示 + 打印


所有页 + 文字水印 + 显示 + 打印



4.调用clearTrackInfo清除水印或条码设置。



注意：应用中需要完成插件初始化后打开文件并加水印的一连串动作，需灵活运用打开的事后监听ocx.registListener("f_open","openPerformed",true)，详见js代码
